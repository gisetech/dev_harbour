Str() — Função. Recebe um número e devolve uma string (Str(3.5) retorna "3.50", por exemplo); você usa o retorno dela, geralmente numa atribuição ou concatenação.

QOut() — Procedimento. Ela executa uma ação (imprimir na tela) e não devolve nada para você usar depois.

Date() — Função. Retorna a data atual como um valor do tipo data, que você guarda numa variável (dHoje := Date()).

Len() — Função. Retorna um número (o tamanho), usado em comparações ou cálculos (IF Len(cNome) > 0).

Upper() — Função. Retorna uma nova string em maiúsculas, sem alterar a original — você precisa capturar esse retorno (cNome := Upper(cNome)).

ClearScreen() — Procedimento. Só executa uma ação (apagar a tela); não há valor de retorno para aproveitar.
